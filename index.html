Bookmarklets

Drag these links to your bookmarks bar to use the bookmarklets.

Full Credit This bookmarklet will give full credit for the current assignment.
Assignment Feedback This bookmarklet will open a feedback panel, to save and inject commonly used feedback.
Feedback Export V1 This bookmarklet is for anyone using the old feedback bookmarklet. It will export all of your feedback into a format that can be imported into the new assignment feedback bookmarklet.
Update Link Dates This bookmarklet is a helper for updating Blackboard content links that use a weekly format, eg: Week 1 (1/11 - 1/16).
github.io Redirect This bookmarklet will toggle between a github.io website and the corresponding github repository.
Display Secret Message This bookmarklet will find and display the secret message for JavaScript Assignment 4.
Open Week Tabs This bookmarklet open the weeks in blackboard as new tabs. It only works for links that start with "Week".
Note: You will probably need to "allow pop-ups" when you use this for the first time.
